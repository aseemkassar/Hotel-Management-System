# 🏨 Hotel Management System (Python)

## 📌 Overview

This is a **Hotel Management System** built using Python.
The application helps manage hotel operations such as room booking, customer details, and billing in an easy and efficient way.

## ✨ Features

* Room booking and checkout system
* Customer details management
* Room availability tracking
* Billing system
* User-friendly GUI interface

## 🛠️ Technologies Used

* Python
* Tkinter (GUI)
* SQLite (Database)
